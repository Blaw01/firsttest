Google Cloud Platform (GCP) Overview

Welcome to the Class 5.5 Armageddon project! This page provides basic information about Google Cloud Platform (GCP).

What is GCP?

Google Cloud Platform (GCP) is a suite of cloud computing services offered by Google. It provides a range of services including computing, storage, and networking, which run on the same infrastructure that Google uses internally for its end-user products.

Key Services

  • Compute Engine: Virtual machines running in Google's data centers.
  • Cloud Storage: Scalable and secure object storage service for unstructured data.
  • BigQuery: Fully-managed, serverless data warehouse for large-scale data analytics.
  • Cloud Pub/Sub: Messaging service for event-driven systems and real-time analytics.

Regions and Zones

GCP resources are hosted in multiple locations worldwide. These locations are divided into regions and zones. A region is a specific geographical location where you can host your resources. Each region has one or more zones, which are isolated locations within the region.
